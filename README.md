# Discord-Mimicker
Mimics every single message of a certain user

________

# MUST READ 

**How to use:
1. Clone by `git clone https://github.com/r3eem/Discord-Mimicker/`
2. Edit the .env file to match your needs :
    A. Put your token in TOKEN
    B. ID of the user you wanna mimic
    C. Put "dm" or "channel", location of sending
    D. Channel ID, leave blank for DM.
    E. Time Delay in Millisecond.
.
